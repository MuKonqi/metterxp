# Kurulum
Öncelikle bilgisayarınızda kurulumun başlaması için şunların olması gerekmektedir: Git, Python3, sudo/su/doas (genellikle sudo kullanılır)<br />
Eğer Debian GNU/Linux kullanıyorsanız 'sudo apt install git python3 -y',<br />
Eğer Fedora Linux kullanıyorsanız 'sudo dnf install git python3 -y',<br />
Eğer Solus kullanıyorsanız 'sudo eopkg install git python3 -y' yazınız.<br />
Eğer sudo kullanıyorsanız 'git clone https://github.com/MuKonqi/metterxp.git ; cd metterxp ; sudo python3 apiutaller.py' yazınız.<br />
Eğer su kullanıyorsanız 'git clone https://github.com/MuKonqi/metterxp.git ; cd metterxp ; su' yazınız. Sonra 'cd /home/kullaniciadiniz/metterxp ; python3 apiutaller.py' yazınız. Not: kullanıcıadiniz yerine kullanıcı adı gelmelidir!<br />
Eğer doas kullanıyorsanız 'git clone https://github.com/MuKonqi/metterxp.git ; cd metterxp ; doas python3 apiutaller.py' yazınız.

# Kaldırma
MetterXP'i kaldırmanın apiutaller kullanmadan 2 yolu vardır:

Birinci yöntem:
1. MetterXP'ı başlatın.
2. Üstteki gri arkaplanlı "MetterXP" yazısına tıklayın.
3. "MetterXP'ı kaldır" tuşuna tıklayın.
4. Şifrenizi girin ve "Onayla" tuşuna tıklayın.
5. "Evet." tuşuna tıklayın.
6. Bu kadar!

İkinci yöntem:
1. MetterXP'ı başlatın.
2. "MetterXP seçenekleri" butonuna tıklayın.
3. Şifrenizi girin ve "Onayla" tuşuna tıklayın.
4. "MetterXP'ı kaldır" butonuna tıklayın.
5. "Evet." butonuna tıklayın.
6. Bu kadar!

# Güncelleme
MetterXP'i 2 yolla güncelleyebilirsiniz:

Birinci yöntem:
1. MetterXP'ı başlatın.
2. Üstteki gri arkaplanlı "MetterXP" yazısına tıklayın.
3. "MetterXP'ı güncelle" tuşuna tıklayın.
4. Şifrenizi girin ve "Onayla" tuşuna tıklayın.
5. "OK" tuşuna tıklayın, bekleyin, tekrar 'OK' tuşuna tıklayın.
6. Bu kadar!

İkinci yöntem:
1. MetterXP'ı başlatın.
2. "MetterXP seçenekleri" butonuna tıklayın.
3. Şifrenizi girin ve "Onayla" tuşuna tıklayın.
4. "MetterXP'ı güncelle" butonuna tıklayın.
5. "OK" tuşuna tıklayın, bekleyin, tekrar 'OK' tuşuna tıklayın.
6. Bu kadar!
