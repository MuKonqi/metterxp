# Installation
First of all, you need the following to start the installation on your computer: Git, Python3, sudo/su/doas (usually sudo is used)<br />
If you are using Debian GNU/Linux 'sudo apt install git python3 -y',<br />
If you are using Fedora Linux 'sudo dnf install git python3 -y',<br />
If you are using Solus, type 'sudo eopkg install git python3 -y'.<br />
If you are using sudo 'git clone https://github.com/MuKonqi/metterxp.git ; cd metterxp ; sudo python3 apiutaller.py'<br />
If you are using su 'git clone https://github.com/MuKonqi/metterxp.git ; cd metterxp ; su'. Then 'cd /home/yourusername/metterxp ; sudo python3 apiutaller.py'. Note: your username must be replaced by a username!<br />
If you are using doas 'git clone https://github.com/MuKonqi/metterxp.git ; cd metterxp ; doas python3 apiutaller.py'.

# Uninstallation
There are 2 ways to uninstall MetterXP without using apiutaller:

First method:
1. Start MetterXP.
2. Click on "MetterXP" with gray background at the top.
3. Click the "Uninstall MetterXP" button.
4. Enter your password and click the "Confirm" button.
5. Click "Yes." button.
6. That's it!

Second method:
1. Start MetterXP.
2. Click the "MetterXP options" button.
3. Enter your password and click the "Confirm" button.
4. Click the "Uninstall MetterXP" button.
5. Click "Yes." button.
6. That's it!

# Update
You can update MetterXP in 2 ways:

First method:
1. Start MetterXP.
2. Click on "MetterXP" with gray background at the top.
3. Click the "Update MetterXP" button.
4. Enter your password and click the "Confirm" button.
5. Click 'OK' button, wait, click 'OK' button again.
6. That's it!

Second method:
1. Start MetterXP.
2. Click the "MetterXP options" button.
3. Enter your password and click the "Confirm" button.
4. Click the "Update MetterXP" button.
5. Click 'OK' button, wait, click 'OK' button again.
6. That's it!
